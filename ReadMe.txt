## Variance-based Multi-output Theta Criterion Adaptive Sampling

This repository contains the source code for the illustrative and engineering examples related to the study of Variance-based Multi-output Theta Criterion Adaptive Sampling. The code implements adaptive sampling methods for multi-output surrogate modeling based on Polynomial Chaos Expansion (PCE). The implementation relies on the open-source Python library UQpy.



## Requirements

The code is written in Python and provided mainly as Jupyter notebooks.



## Install the main dependencies with:

UQpy:
https://uqpyproject.readthedocs.io/
